# Backend of the project

## How to run the backend

1. Make venv using python

### `python -m venv venv`

2. Activate the venv

2.1. For Linux

### `source venv/bin/activate`

2.2. For Windows

### `venv\Scripts\activate`

3. Install the requirements

### `pip install -r requirements.txt`

4. Run the backend

### `python app.py`
